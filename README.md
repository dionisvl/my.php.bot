# TG bot PHP example

```
cp config.example.php config.php
```

- link to setup webhook url:
```
https://api.telegram.org/botYOUR_TG_SECRET_KEY/setWebhook?url=https://YOUR_WEBHOOK_URL
```
(https is required AND <bot> before secret is required)
